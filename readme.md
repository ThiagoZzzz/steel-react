# STEEL  - REACT

Este repositorio contiene el proyecto final/entregable realizado para el curso de Desarrollo Web NUCBA, respectivo al módulo de React. Se trata de una tercera iteración sobre la aplicación web STEEL adaptada completamente al framework de React.

* Navegación.
* Renderizado dinámico de elementos.
* Carrito de compras.
* Carrusel de productos.
* Filtros de productos.
* Autenticación.
* Validación de formularios.

🔗 **[Demo (Vercel)](https://steel-react.vercel.app/)**

---

## 🛠️ Tecnologías Utilizadas

* **Frontend:** React, JavaScript.
    * Styled-components.
    * Phosphor Icons.
    * React Router.
    * Context API.
    * Custom Hooks.
    * React Hook Form.
    * Zod.
* **Despliegue:** Vercel.

---

## ✨ Características Principales

* **Diseño Mobile First:** Totalmente responsivo, adaptándose desde pantallas móviles hasta monitores de escritorio.
* **Interfaz Moderna:** Con Styled-components se implementó un CSS moderno (Grid, Flexbox, y efectos como Glassmorphism).
* **Funcionalidades:** Mediante React y sus herramientas se implementó el manejo de estados y la interactividad de la aplicación (navegación, filtros, manejo de contexto, validación de formularios, entre otros).
